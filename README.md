# Music Edu
MEAN stack

## Get started
```
git clone https://github.com/OumaimaTiguint/music-edu.git
```
### front-end
```
npm install
```

```
ng serve
```
### Server side

```
cd server
```
```
npm install
```
```
npm i nodemon
```
```
nodemon index.js
```

## Open in browser
Go to http://localhost:4200/
